# Studytime Check App
---

### 프로젝트 설명

### StudytimeCheck App 사용 방법
1. Studytime Measuring
- SPACEBAR를 입력하면 측정을 시작하고 다시 한 번 SPACEBAR를 입력하면 측정을 종료합니다.
- 측정이 종료되면 공부 기록이 저장되어 통계나 그룹 랭킹도 볼 수 있습니다.

2. Displaying stats


### 팀원
1. 강봉수(20학번, 글로벌소프트웨어융합전공)
2. 김근희(22학번, 글로벌소프트웨어융합전공)
3. 문소정(21학번, 심화컴퓨터전공)
